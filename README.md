# Retro Shooter

A retro-style top-down shooter built with vanilla HTML5 Canvas. No installs, no downloads — just open and play.

**[Play it here](https://jordanrodz.github.io/retro-shooter)**

---

## How to Play

Survive endless waves of enemies, collect XP gems, level up, and see how far you can go.

| Control | Action |
|---|---|
| WASD / Arrow Keys | Move |
| Mouse | Aim |
| Left Click (hold) | Shoot |
| M | Mute / Unmute |

**On mobile:** left thumb to move (virtual joystick), right thumb to aim and shoot.

## Features

- 4 enemy types — Grunts, Scouts, Tanks, and Bosses every 5th wave
- 12 permanent upgrades to choose from on level up (Spread Shot, Piercing, Explosive, Lifesteal, and more)
- 4 powerup drops — HP, Speed Boost, Rapid Fire, Shield
- Fully synthesized retro audio via Web Audio API
- Works on desktop and mobile
